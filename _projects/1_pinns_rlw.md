Built a custom PINN framework in PyTorch to solve the Regularized Long-Wave (RLW) equation. The model was validated against traditional numerical solvers, achieving high-fidelity results that demonstrate the power of scientific machine learning for nonlinear wave equations.

Key achievements:
- Custom loss function incorporating PDE residual, initial conditions, and boundary conditions
- Adaptive learning rate scheduling for improved convergence
- Comparison with finite difference methods showing competitive accuracy
